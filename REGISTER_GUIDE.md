# 📝 注册功能完整使用指南

## 🎯 功能特性

✅ **邮箱验证码注册** - 需要验证邮箱所有权  
✅ **实时表单验证** - 前端即时验证输入  
✅ **友好的用户提示** - 清晰的错误和成功消息  
✅ **倒计时保护** - 60 秒防重复发送  
✅ **安全密码要求** - 8-32 位密码强度控制  
✅ **验证码保护** - 5 分钟有效期，一次性使用  

## 🚀 使用步骤

### 步骤 1: 访问注册页面

```
http://localhost:3000/register
```

### 步骤 2: 填写昵称

在第一个输入框中输入您的昵称：
- **要求**: 2-20 个字符
- **示例**: `张三`, `AI Explorer`, `TechLover`

### 步骤 3: 输入邮箱并获取验证码

1. 在"邮箱地址"输入框中输入您的邮箱
   ```
   示例：user@example.com
   ```

2. 点击右侧的 **"获取验证码"** 按钮

3. 此时会发生：
   - ✅ 调用后端 API 发送邮件
   - ✅ 按钮进入 60 秒倒计时
   - ✅ 显示成功提示："✓ 验证码已发送，请检查邮箱（有效期 5 分钟）"

4. **检查邮箱**
   - 打开您的邮箱
   - 查找来自 "Hot AI" 的邮件
   - 如果没找到，请检查垃圾邮件文件夹
   - 邮件包含 6 位数字验证码

### 步骤 4: 输入验证码

在"验证码"输入框中输入收到的 6 位数字：
```
示例：123456
```

**注意**: 
- ✅ 必须是 6 位数字
- ✅ 必须在 5 分钟内使用
- ✅ 只能使用一次

### 步骤 5: 设置密码

在"密码"输入框中设置您的密码：
- **最少 8 位**
- **最多 32 位**
- **建议**: 包含字母和数字组合

```
✅ 好密码示例: SecurePass123!, MyPassword2024
❌ 弱密码示例：12345678, password, abcdefgh
```

### 步骤 6: 确认密码

在"确认密码"输入框中再次输入相同的密码。

### 步骤 7: 提交注册

点击 **"立即注册"** 按钮

**成功后**:
- ✅ 显示绿色提示："注册成功！正在跳转到登录页..."
- ✅ 自动跳转到登录页面
- ✅ 可以使用新账号登录

```typescript
// 1. 邮箱格式验证
const isValidEmail = (email: string): boolean => {
  const emailRegex = /^[^\s@]+@[^\s@]+\.[^\s@]+$/
  return emailRegex.test(email)
}

// 2. 验证码验证（6 位数字）
if (!/^\d{6}$/.test(verificationCode)) {
  error.value = '请输入 6 位数字验证码'
}

// 3. 密码验证
if (password.length < 8 || password.length > 32) {
  error.value = '密码长度必须在 8-32 位之间'
}

if (password !== confirmPassword) {
  error.value = '两次输入的密码不一致'
}
```

## ⚠️ 常见问题

### Q1: 为什么收不到验证码？

**可能原因**:
1. 邮箱地址输入错误
2. 邮件被归类到垃圾邮件
3. 邮件服务器延迟
4. 触发限流保护

**解决方案**:
```
✅ 检查邮箱地址是否正确
✅ 查看垃圾邮件文件夹
✅ 等待 1-2 分钟
✅ 60 秒后重新发送
```

### Q2: 验证码无效怎么办？

**可能原因**:
1. 验证码已过期（超过 5 分钟）
2. 验证码已被使用
3. 输入错误

**解决方案**:
```
✅ 重新获取新的验证码
✅ 确保在有效期内使用
✅ 仔细核对 6 位数字
```

### Q3: 提示"该邮箱已注册"怎么办？

**原因**: 该邮箱已经注册过账号

**解决方案**:
```
✅ 使用其他邮箱注册
✅ 直接去登录页面
✅ 如果忘记密码，使用找回密码功能
```

### Q4: 提示"操作过于频繁"怎么办？

**原因**: 短时间内多次请求验证码

**解决方案**:
```
✅ 等待 1 分钟后再试
✅ 避免连续点击按钮
✅ 遵守 60 秒间隔限制
```

### Q5: 密码有什么要求？

**密码规则**:
```
✅ 最少 8 位
✅ 最多 32 位
✅ 建议包含字母和数字
✅ 区分大小写
```

**推荐密码**:
```
✅ StrongPass123!
✅ MySecure2024
✅ HotAI@User888
```

**不推荐密码**:
```
❌ 12345678
❌ password
❌ abcdefgh
❌ 88888888
```
