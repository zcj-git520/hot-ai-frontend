# 管理后台重构设计方案

## 1. 概述

重新设计管理后台，采用侧边栏 + 内容区布局，包含6个主要功能模块。

### 标签页结构
1. **Dashboard** - 报表概览 + 各模块待审核数量
2. **课程管理** - 列表 + CRUD + 发布/下架 + 章节管理
3. **工具管理** - 全部列表 + 状态管理 + 审核操作
4. **文章管理** - 列表 + 上下架 + 审核操作
5. **职业管理** - 列表 + CRUD + 风险评估
6. **用户管理** - 列表 + 角色管理 + 操作日志

## 3. 侧边栏设计

### 视觉风格
- 固定宽度 240px
- 深色科技风格背景（#1a1a2e 或类似）
- 导航项：图标 + 名称
- 选中态：高亮背景色
- 徽章：显示待审核数量

### 导航项
| 图标 | 名称 | 徽章 |
|------|------|------|
| 📊 | Dashboard | - |

| 📝 | 文章管理 | 待审核数量 |

### 底部
- 管理员头像/名称
- 退出按钮

## 4. 功能模块设计

### 4.1 Dashboard（概览）
**功能**：
- 统计卡片：课程总数、工具总数、文章总数、用户总数
- 待审核汇总：工具待审核数、文章待审核数
- 快捷入口：点击可跳转到对应管理模块

**布局**：卡片网格 + 待审核列表

### 4.2 课程管理
**功能**：
- 课程列表（搜索、难度筛选、状态筛选）
- 创建课程（弹窗表单）
- 编辑课程（弹窗表单）
- 删除课程（确认对话框）
- 发布/下架
- 设为精选/取消精选
- 章节管理（弹窗内）

**布局**：卡片网格（与 /admin/learning-paths 一致）

### 4.3 工具管理
**功能**：
- 全部工具列表（搜索、状态筛选）
- 查看工具详情
- 审核操作：通过 / 拒绝 / 要求修改
- 状态管理：上线/下线

**布局**：列表 + 详情侧滑/弹窗

### 4.4 文章管理
**功能**：
- 文章列表（搜索、分类筛选、状态筛选）
- 审核操作：通过 / 拒绝
- 上下架管理

**布局**：列表 + 操作按钮

### 4.5 职业管理
**功能**：
- 职业列表（搜索、难度筛选）
- 创建职业（弹窗表单）
- 编辑职业（弹窗表单）
- 删除职业（确认对话框）
- 风险评估管理

**布局**：列表 + 弹窗表单

### 4.6 用户管理
**功能**：
- 用户列表（搜索、角色筛选）
- 角色管理（管理员/普通用户）
- 禁用/启用用户
- 操作日志查看

**布局**：列表 + 详情弹窗

## 5. 技术实现

### 组件结构
```
pages/admin/
├── index.vue              # 主页面（侧边栏 + 内容区）
└── components/
    ├── AdminSidebar.vue   # 侧边栏组件（重构）
    ├── dashboard/
    │   └── DashboardContent.vue
    ├── courses/
    │   └── CoursesContent.vue
    ├── tools/
    │   └── ToolsContent.vue
    ├── articles/
    │   └── ArticlesContent.vue
    ├── professions/
    │   └── ProfessionsContent.vue
    └── users/
        └── UsersContent.vue
```

### API复用
- `adminApi.learningPath` - 课程管理
- `adminApi.tool` - 工具管理
- `articleApi` - 文章管理（需确认是否有admin接口）
- `professionApi` - 职业管理（需确认是否有admin接口）
- 新增 `adminApi.user` - 用户管理

### 状态管理
- 当前选中侧边栏项（`activeMenu`）
- 各模块数据（组件内部state）
- 通用：toast通知、loading状态
